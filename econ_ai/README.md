# Coding with AI (for Economics Students)

Setup and writing guides for Clark University Economics courses `ECON206` and `ECON10`.
Part of the [Teaching hub](https://kensukesuzuki04.github.io/teaching/).

## Start Here

1. [Pre-Course Preparation](setup_pre_course_preparation.md) — **do this first**; includes GitHub Student Pack & Copilot activation (approval can take a few days), then VS Code, Python, Quarto, and a PDF engine.
2. [Quarto Drafting Guide](setup_quarto_draft_guide.md) — how to write figures, tables, equations, and render to PDF.
3. [How to Use GitHub Copilot](setup_student_copilot_instructions.md) — responsible, effective use of Copilot in coursework.
4. [Token-Saving for Claude Code](setup_token_saving_claude_code.md) — *advanced/optional*: RTK + Caveman setup to reduce token usage (Claude Code CLI on Windows).
